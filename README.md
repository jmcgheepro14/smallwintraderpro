# SmallWin Trader Complete Prototype

A subscription-ready beginner trading education app prototype.

## Included screens
- Login
- Create account
- Dashboard
- Best setup
- Live scanner
- Trade coach
- Paper journal
- Upgrade/pricing
- Demo checkout
- Account page
- Legal pages

## Important
This prototype uses in-session demo authentication and demo payments. For production, connect Firebase/Supabase/Auth0 and Apple In-App Purchases, Google Play Billing, or Stripe Checkout for web.

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
