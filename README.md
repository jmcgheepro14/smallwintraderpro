# SmallWin Trader

Polished beginner trading education and practice app.

Includes live stock scanning, beginner setup scoring, entry zones, stop losses, profit targets, account-size based sizing, paper journal, subscription pricing screen, free/pro demo logic, legal placeholder pages, and app-store readiness notes.

## Important
Educational prototype only. Not financial advice. Have an attorney review before public launch.

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Deploy
Main file path: `app.py`

## Mobile App Note
For Apple App Store and Google Play Store, rebuild as React Native or Flutter and connect Apple In-App Purchases, Google Play Billing, authentication, backend database, privacy URL, support URL, and terms URL.
