# SmallWin Trader Pro

A polished beginner trading coach app built with Streamlit.

## Features

- Dark fintech style design
- Live stock scanner
- Beginner-friendly setup grading
- Entry zones
- Stop losses
- Profit targets
- Account-size based share sizing
- Paper trade journal
- Beginner lessons

## Important

This is an educational practice app, not financial advice.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy

Upload these files to GitHub and deploy through Streamlit Community Cloud.

Main file path:

```bash
app.py
```
